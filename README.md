## 概要

ハンターハンターのキャラクターのセリフを返す bot です。
強化系、変化系、具現化系、放出系、操作系、特質系の中から１つを選択し、LINE 公式アカウントに送信するとその系統のキャラクターのセリフが返ってきます。
管理者用サイトでキャラクターの追加、セリフの追加ができます。
